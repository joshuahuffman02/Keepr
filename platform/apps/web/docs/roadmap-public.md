# Public Roadmap

## Now

- POS & store reliability: offline queue + sync log, charge-to-site hardening, refunds/credits, gift card balance view.
- Guest experience: portal timelines (messages, orders), delivery/curbside options, “assign on arrival” booking choice.
- Trust & compliance: audit log, permissions by role, approval flows, privacy/PII settings.
- OTA iCal: per-listing calendar export/import with sync status and error badges.
- Finance: payouts/disputes views, deposits v2, ledger/tax sanity checks.

## Next

- Two-way OTA (availability/rates/reservations) with logs, retry, and per-channel status.
- Support & Ops: unified comms timeline, SLA/overdue badges, ticketing, and staff chat.
- Reporting: pickup/pace dashboards, saved views, CSV/S3 exports, warehouse connector stub.
- Integrations wave 1: accounting (payout/GL export), CRM/helpdesk, access control stubs.
- PWA/offline: queued orders/payments/messages, push notifications, visible sync state.

## Later

- Public API & webhooks, OAuth2 sandbox, SDKs.
- AI assist: replies, task bundling, routing, smart search, anomaly alerts.
- IoT hooks: metering/lock/alert stubs; QR-at-site flows.
- Enterprise/international: multi-property controls, approvals depth, localization/currency/tax packs.
