# Local Development

Start the API and Web dev servers.

## Usage

$ARGUMENTS

### Full stack
```bash
pnpm dev
```

### API only (port 4000)
```bash
pnpm api
```

### Web only (port 3000)
```bash
pnpm web
```

## Notes
- Ensure `pnpm install` has been run.
- Populate `.env` with required values for local dev.
