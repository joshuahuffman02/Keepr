## Exit Interview Security Items

Remind departing employee:

- [ ] Confidentiality obligations continue after employment
- [ ] Return all company information and materials
- [ ] Delete any company data from personal devices
- [ ] Do not share passwords or access with anyone
- [ ] Non-compete/non-solicitation terms (if applicable)

---

## Special Circumstances

### Involuntary Termination

If termination is involuntary, complete these steps BEFORE notifying employee:

- [ ] Coordinate timing with IT to disable access during meeting
- [ ] Have equipment ready to collect
- [ ] Arrange escort if needed
- [ ] Document any concerning behavior

### Remote Employee

Additional steps for remote workers:

- [ ] Arrange equipment return shipping
- [ ] Provide pre-paid shipping label
- [ ] Set deadline for equipment return
- [ ] Wipe devices remotely if possible
