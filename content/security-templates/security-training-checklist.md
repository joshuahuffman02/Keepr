### 2. Phishing & Email Security (20 minutes)

| Topic | Covered | Notes |
|-------|---------|-------|
| [ ] What is phishing? | | |

| [ ] Checking sender addresses | | |
| [ ] Hovering over links before clicking | | |
| [ ] Reporting suspicious emails | | |
| [ ] Business email compromise | | |
| [ ] Spear phishing (targeted attacks) | | |

**Red Flags to Watch For:**
- Urgent or threatening language
- Unexpected attachments
- Requests for passwords or personal info
- Misspelled sender domains (amaz0n.com vs amazon.com)
- Generic greetings ("Dear Customer")

**Practice Exercise:** Show 5 sample emails and have employees identify which are phishing.

## Training Assessment

### Quiz Questions (minimum 80% to pass)

1. What should you do if you receive an unexpected email asking you to click a link?
2. How long should your password be at minimum?
3. Who should you contact if you think you clicked on a phishing link?
4. True/False: It's okay to share your password with a coworker if they need access.
5. What should you do before leaving your workstation?
