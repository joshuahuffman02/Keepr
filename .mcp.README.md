# MCP Server Configuration

This project supports Model Context Protocol (MCP) servers for extended functionality.

## Current Status

- `.mcp.json` - Active configuration (currently empty)
- `.mcp.example.json` - Example configurations for reference

## Available MCP Servers

### 1. **Slack**

Enable Slack integration for notifications and team communication.

**Requirements:**

- `SLACK_BOT_TOKEN` - Get from https://api.slack.com/apps
- `SLACK_TEAM_ID` - Your Slack workspace ID

**Enable:** Copy the `slack` block from `.mcp.example.json` to `.mcp.json` and add your tokens.

---

### 2. **PostgreSQL**

Direct database queries for debugging and data exploration.

**Requirements:**

- PostgreSQL connection string (uses `DATABASE_URL` from environment)

**Enable:** Copy the `postgres` block and update the connection string.

---

### 3. **GitHub**

GitHub repository integration for issues, PRs, and code management.

**Requirements:**

- `GITHUB_PERSONAL_ACCESS_TOKEN` - Generate at https://github.com/settings/tokens

**Enable:** Copy the `github` block and add your token.

---

### 4. **Filesystem**

Extended filesystem access beyond the project directory.

**Enable:** Copy the `filesystem` block and specify allowed directories.

---

### 5. **Sentry**

Access Sentry error logs directly from Claude.

**Requirements:**

- `SENTRY_AUTH_TOKEN` - Get from Sentry settings
- `SENTRY_ORG` - Your organization slug
- `SENTRY_PROJECT` - Project name

**Enable:** Copy the `sentry` block and configure credentials.

---

### 6. **Brave Search**

Web search capabilities for research and information gathering.

**Requirements:**

- `BRAVE_API_KEY` - Get from https://brave.com/search/api/

**Enable:** Copy the `brave-search` block and add your API key.

---

## How to Enable an MCP Server

1. Choose a server from `.mcp.example.json`
2. Copy the entire server configuration block
3. Paste it into `.mcp.json` under `mcpServers`
4. Configure required environment variables
5. Restart Claude Code

**Example:**

```json
{
  "$schema": "https://modelcontextprotocol.io/schema.json",
  "mcpServers": {
    "postgres": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-postgres",
        "postgresql://localhost:5432/campreserv"
      ]
    }
  }
}
```

## Security Notes

- Never commit API keys or tokens to git
- Use environment variables for sensitive data
- Add `.mcp.json` to `.gitignore` if it contains secrets
- MCP servers run with your user permissions
