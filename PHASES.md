# Phases

## Phase 1: Pricing & Payments
Done when pricing plans are defined, a purchase flow exists for the chosen payment provider, and tests cover pricing and checkout logic.

## Phase 2: Operations
Done when core operational workflows (e.g., task tracking, staff workflows, admin tooling) are in place with tests and documented runbooks.

## Phase 3: Analytics
Done when analytics surfaces key metrics, supports reporting/export, and validates data integrity with automated checks.

## Phase 4: Automation
Done when automation rules and scheduled jobs can be configured, monitored, and audited safely.

## Phase 5: Finalization & UX polish
Done when UX polish, performance, and accessibility passes are complete and documentation is production-ready.
