# UI Development (Codex)

## Use when
- Creating or modifying React components and styles.

## Core patterns
- TypeScript props and `cn()` for class composition.
- Semantic color tokens (emerald primary, slate neutrals).
- Mobile-first layouts and consistent spacing.

## Checklist
- Component responsibilities are clear.
- Focus/hover/loading/empty states exist.
- Responsive breakpoints are tested.

## Allowed scope
- UI guidance only unless asked to implement.

## Ask before proceeding if unclear
- The target component or design tokens are unknown.

## Stop condition
- A clear UI implementation plan is provided.
