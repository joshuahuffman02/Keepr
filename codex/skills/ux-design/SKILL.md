# UX Design (Codex)

## Use when

- Designing flows or improving usability for Campreserv roles.

## User types

- Guests: booking and reservation management.
- Staff: check-in/out and payments.
- Admins: settings and reporting.

## Principles

- Speed over polish for staff tasks.
- Error prevention and clear recovery paths.
- Progressive disclosure and consistent feedback.

## Patterns

- Clear CTAs, search/filter, bulk actions, pagination.
- Mobile touch targets and simplified layouts.

## Allowed scope

- UX guidance only unless asked to implement.

## Ask before proceeding if unclear

- The user role or primary task is unspecified.

## Stop condition

- UX recommendations and risks are listed.
